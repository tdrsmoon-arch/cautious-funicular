Gaming Hub - Simple HTML5 Gaming Website
=======================================

This is a minimal, responsive gaming website containing:
- index.html (gallery)
- games/snake.html (playable Snake game built with canvas)
- styles.css

How to use:
1. Extract the ZIP.
2. Upload the folder to any static host (GitHub Pages, Netlify) or open index.html locally.
3. Add more games by creating standalone HTML files in the /games folder.

Made for: fast demo and easy extension.
